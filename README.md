# Unit Converter using Python

A command-line Unit Converter built using Python. This project allows users to convert values between different measurement units, including temperature, distance, weight, and length.

The application provides a simple menu-driven interface that makes conversions quick, accurate, and easy to use.

## 📌 Project Overview

This project is designed to simplify everyday unit conversions through an interactive terminal application.

Users can select a conversion type from the menu, enter a value, and instantly receive the converted result.

The application continuously runs until the user chooses to exit.

This project is suitable for beginners who want to practice Python fundamentals and build interactive command-line applications.

## 🔄 Supported Conversions

### 🌡️ Temperature

* Celsius → Fahrenheit
* Fahrenheit → Celsius

### 📏 Distance

* Kilometers → Miles
* Miles → Kilometers

### ⚖️ Weight

* Kilograms → Pounds
* Pounds → Kilograms

### 📐 Length

* Meters → Feet
* Feet → Meters
