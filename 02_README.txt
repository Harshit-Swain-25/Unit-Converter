==============================
       Unit Converter
==============================

Description:
A Python program to convert units such as
temperature, distance, weight, and length.

Conversions:
- Celsius <-> Fahrenheit
- Kilometers <-> Miles
- Kilograms <-> Pounds
- Meters <-> Feet

How to Run:
1. Save the file as 01_main.py
2. Open VS Code terminal.
3. Run the program with:
   python3 01_main.py
